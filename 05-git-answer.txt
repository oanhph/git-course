Ex1. Giải thích câu lệnh: 
- git commit --amend: Thay đổi commit msg 
    - i: Vào chế độ insert 
    - esc: Thoát insert 
    - :wq: write & quit 
-  git reset HEAD~3: Đưa 3 commit từ vùng repository về working directory (uncommit)
